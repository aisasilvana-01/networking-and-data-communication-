# networking-and-data-communication-
This repository contains documentation of software configuration used in a network system based on Ubuntu Server. The configured services include:

1. DHCP Server: Automatically assigns IP addresses to clients connected to the network
2. NAT (Network Address Translation): Translates private IP addresses to public IP addresses and vice versa
3. DNS Server: Manages domain names so clients can access services using domain names instead of IP addresses
4. Web Server: Hosts and serves web applications or websites
5. SMTP/Mail Server: Handles sending and receiving emails
6. FTP Server: Manages file transfer between client and server

To explore the configurations that have been implemented, you can download the provided virtual machine. It includes an Ubuntu Server (as the main server) and an Ubuntu Desktop (as the client).

All configurations were performed using VMware Workstation 17 Pro. It is recommended to run the virtual machine using the same or a newer version of VMware to ensure compatibility.

This repository can be used as a practical reference for learning how to build and configure a complete network service environment in a virtualized setup.
